Arcana Exchange

О проекте

Arcana Exchange - это инструмент для поиска игроков для обмена карт Лунных Аркан в Genshin Impact.

Как это работает

  1. Вы загружаете свои данные из HoYoLAB.
  2. Сайт анализирует вашу коллекцию.
  3. Показывает игроков, которые ищут ваши лишние карты или у которых есть нужные вам карты.
  4. Обмен происходит внутри игры.

Безопасность

Сайт не требует:

  • пароль HoYoLAB;
  • cookie;
  • токены авторизации.

Для обновления коллекции используются:

  • экспорт JSON или HTML из ресурсов HoYoLAB;
  • публичные игровые данные;
  • verification через подпись профиля.
